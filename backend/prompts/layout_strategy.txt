You are an expert CSS layout architect. Given the following Figma design node hierarchy with their properties, determine the optimal CSS layout strategy for each container node.

For each container node, decide one of:
- "flex" - Use CSS Flexbox (for auto-layout or linear arrangements of children)
- "grid" - Use CSS Grid (for grid-like patterns with rows and columns)
- "absolute" - Use absolute positioning (for overlapping or free-form layouts)
- "block" - Use simple block layout (for single children or simple stacked elements)

Decision guidelines:
1. Auto-layout nodes in Figma should ALWAYS map to flexbox. The auto-layout direction (HORIZONTAL/VERTICAL) maps to flex-direction (row/column). Item spacing maps to gap.
2. Overlapping children (where bounding boxes intersect) require absolute positioning. The parent becomes position: relative and children get position: absolute with explicit top/left/width/height.
3. Children arranged in equal-spaced rows AND columns suggest CSS Grid. Look for consistent column widths and row heights across multiple children.
4. A container with 0 or 1 children should use block layout.
5. Children arranged in a single row (similar Y positions, increasing X) suggest flex with direction: row.
6. Children arranged in a single column (similar X positions, increasing Y) suggest flex with direction: column.
7. Always preserve the visual hierarchy exactly as designed.

For flex decisions, also specify:
- flex_direction: "row" or "column"
- justify_content: maps from Figma's primaryAxisAlign (MIN→flex-start, CENTER→center, MAX→flex-end, SPACE_BETWEEN→space-between)
- align_items: maps from Figma's counterAxisAlign (MIN→flex-start, CENTER→center, MAX→flex-end, BASELINE→baseline)
- gap: from Figma's itemSpacing (e.g., "8px")
- flex_wrap: "wrap" if Figma's layoutWrap is "WRAP"

For grid decisions, also specify:
- grid_template_columns: e.g., "repeat(3, 1fr)" or "200px 200px 200px"
- gap: spacing between grid cells

Return a JSON object where each key is a node ID and the value is an object with:
{
  "strategy": "flex|grid|absolute|block",
  "flex_direction": "row|column" (if flex),
  "justify_content": "..." (if flex),
  "align_items": "..." (if flex),
  "gap": "..." (if applicable),
  "grid_template_columns": "..." (if grid),
  "grid_template_rows": "..." (if grid),
  "notes": "brief reasoning"
}

Return ONLY valid JSON, no other text.
